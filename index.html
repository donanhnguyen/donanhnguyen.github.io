TransplantCentral

TransplantCentral is a web application that allows you to review your favorite US metropolitan cities based of a variety of categories and share your living/travel experiences.

- Ruby on Rails MVC architecture was used for developing and designing CRUD features of the application, allowing users to create accounts and post, delete posts, and edit posts, utilizing PostgreSQL database.

- Vanilla Javascript was used for functions calculating average City's rating, and also average rating per category for a city, best rated city, lowest rated city, etc.

- Utilized ReactJS for developing components and Redux library for managing application state, and Webpack and NPM used for Javascript implementation with Rails.

- Responsive desktop web page design utilizing HTML/CSS without Bootstrap.

Technologies used: Ruby on Rails, Ruby, Javascript (ES5/ES6), ReactJS, Redux, PostgreSQL, HTML, CSS, MVC, Webpack, NPM
