TransplantCentral

TransplantCentral is a web application that allows you to review your favorite US metropolitan cities based of a variety of categories and share your living/travel experiences.

- Ruby on Rails MVC architecture used for developing CRUD features, allowing users to create accounts and post, delete posts, and edit posts, utilizing PostgreSQL database.

- Vanilla Javascript used for functions calculating average City's rating, and also average rating per category for a city, best rated city, lowest rated city, etc.

- ReactJS components and Redux library used for managing application state, and Webpack and NPM used for Javascript implementation with Rails.

- Responsive design with HTML/CSS without Bootstrap.

Technologies used: Ruby on Rails, Ruby, Javascript (ES5/ES6), ReactJS, Redux, PostgreSQL, HTML, CSS, MVC, Webpack, NPM

BlackJack

BlackJack, (also known as "21"), is an intuitive card game between a dealer and several players or only one player. The dealer and player are both dealt 2 cards. The goal is to:

1. Get 21, or as close as possible to 21 without going over.

2. If you go over 21, you lose. If dealer goes over 21, they lose.

3. If neither of you goes over 21, the one with the higher score wins.

- Player can place bet on each hand, with the option to HIT, STAY, or DOUBLEDOWN (Doubles the bet for only one more card). If money is depleted, then game is over. Players can play until their money runs out.

- Technologies Used: Vanilla Javascript (ES5, ES6), HTML, CSS.
